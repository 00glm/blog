Development Update #128

2017/04/21

New Wallet Version:

Skycoin Wallet version 15 is on the website, which fixes dozens of UI bugs. - Transaction status should be working - There is a combined installer for Windows 32⁄64 bit now - The windows path issue was fixed - Peer to peer connections and PEX (peer exchange) are enabled now
